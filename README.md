# SlowExtractionSimulations
This repo contains material for simulating the slow extraction process in the CNAO synchrotron.

## Structure
| *path* | *description* |
| --- | --- |
| `LATTICE` | a collection of reference lattices and optics files |
| `pyLib` | a collection of customised python functions and modules |
| `templates` | a collection of real-life examples and templates |
| `examples_pyLib` | a collection of examples applying the functions of pyLib |
| `FFT_tunes` | a collection of particle tune calculations with FFT |
| `Xsuite_user_manual_examples` | a collection of examples on key features of Xsuite |
