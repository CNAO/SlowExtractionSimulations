# Betatron Core Simulations
A collection of simulations on slow extraction using a Betatron Core to excite the beam.

## File Description
| *file* | *description* |
| --- | --- |
| `Betatron_Core` | simple example of excitation with a custom Betatron core element |
| `Betatron_Core_with_Cavity` | example of excitation of a debunched beam turning off the RF Cavity before activating the Betatron core |
