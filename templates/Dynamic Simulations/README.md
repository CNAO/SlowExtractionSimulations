# Dynamic Simulations
A collection of simulations introducing time-dependent elements.

## File Description
| *file* | *description* |
| --- | --- |
| `Sextupole Ramp-up` | example for sextupole ramping and ivestigation of extracted particles according the momentum offset |
| `RF_Cavity` | example of introducing an RF cavity to the lattice and plotting the longitudinal phase space |
| `Acceleration` | simple example of acceleration of a beam with a chosen energy program |
