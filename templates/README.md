# Template folder
Collection of real-life examples and templates.

## External repos
The templates are based on few external repos which contain either useful python modules and/or lattice/optics data.
Hence, for the templates to correctly work, the external repos must be checked out before.

| *folder* | *description* | *URL* |
| --- | --- | --- |
| `slow_extraction_tutorial` | example of simulating slow extraction as prepared by the XSuite dev team | git@github.com:xsuite/slow_extraction_tutorial.git |
