# RFKO Simulations
A collection of simulations on slow extraction using RFKO. 

## File Description
| *file* | *description* |
| --- | --- |
| `RFKO_linear_freq_chirp` | example of excitation with a linear frequency chirp |
| `RFKO_with_RF_Cavity` | example of excitation of a bunched beam with an RF Cavity ON |
| `RFKO_freq_sawtooth` | example of excitation with a sawtooth frequency sweep |
| `RFKO_sawtooth_with_ripples` | example of excitation with the presence of quadrupole ripples |
| `Sawtooth_Test` | testing and experimenting with the sawtooth signal |
| `RFKO_Sawtooth.py` | example of a simulation with a sawtooth frequency sweep with increased amount of turns and particles |
